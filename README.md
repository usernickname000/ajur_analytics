## Возможности

### 📈 Анализ периода
Загружаешь Excel-выгрузку из CRM — утилита автоматически строит полный отчёт:
- Месячная статистика и динамика выручки
- Топ клиентов и менеджеров
- Анализ по отраслям
- Сезонность и квартальные коэффициенты
- RFM-анализ клиентской базы
- Топ номенклатур
- Прогноз на следующий месяц

### 🔄 Сравнение периодов
Загружаешь два файла за разные периоды — получаешь сравнительный отчёт:
- Выручка и заказы по менеджерам: δ и %
- Топ клиентов с динамикой
- Новые и ушедшие клиенты
- Сравнительные графики

### 🎯 Выполнение плана
- Задаёшь плановые суммы по каждому менеджеру прямо в интерфейсе
- В отчёте появляется лист с выполнением: факт / план / % / статус ✅ ⚠ ❌
- Планы сохраняются автоматически между сессиями

### 👁 Папка-наблюдатель
- Указываешь папку — утилита следит за ней в фоне
- Как только появляется новый `.xlsx` файл, через 5 минут автоматически запускается анализ
- Не нужно каждый раз нажимать кнопку вручную

## Установка и запуск

**1. Клонировать репозиторий:**
```powershell
git clone https://github.com/usernickname000/ajur_analytics.git
cd ajur_analytics
```

**2. Установить зависимости:**
```powershell
python -m pip install -r requirements.txt
```

**3. Запустить:**
```powershell
python app.py
```

---

## Компиляция в .exe

```powershell
python -m pip install pyinstaller
python -m PyInstaller --onefile --windowed --icon=icon.ico --add-data "icon.ico;." --name "Аналитика_Фонтанка" app.py
```

Готовый файл появится в папке `dist\`.

> ⚠ При первом запуске Windows Defender может показать предупреждение — это ожидаемо для неподписанных `.exe`. Нажми «Подробнее → Всё равно запустить».

## Формат входного файла

Утилита работает с выгрузками из CRM в формате `.xlsx`. Обязательные колонки:

| Колонка | Описание |
|--------|---------|
| `Месяц` | Период в формате `ММ.ГГГГ` |
| `Заказ` | Номер заказа |
| `Дата заказа` | Дата |
| `Менеджер` | Имя менеджера |
| `Выручка без НДС` | Сумма в рублях |
| `Проект` | Тип проекта |
| `Клиент РА` / `Клиент` | Название клиента |
