## О проекте

Десктопное приложение для коммерческого отдела. Принимает выгрузку заказов из CRM (`.xlsx`), строит Excel-отчёт с 20+ листами, HTML-дашборд в браузере и сверяет итоги с данными бухгалтерии. Работает полностью офлайн.

---

## Возможности

### Анализ периода
- Месячная выручка с внешними доходами (программатик) и YoY-сравнением
- Топ-50 клиентов и топ-30 менеджеров
- Выручка по отраслям, сезонность, кварталы
- RFM-сегментация: Champions / Loyal Customers / At Risk / Hibernating
- Лояльность клиентов — распределение по количеству заказов
- Топ-50 номенклатур с долей выручки
- Прогноз на следующий месяц (линейный тренд)
- **Анализ скидок** — по клиентам, менеджерам, бакетам (0–5%, 5–10% и т.д.)
- **Автоматические сигналы и аномалии** — месячные выбросы (>2σ), трёхмесячный спад, концентрация клиентов, RFM-риски, разрыв между менеджерами
- **Графики прямо в Excel** — линейный, горизонтальный столбчатый, вертикальный столбчатый, круговая

### Сверка с бухгалтерией
- Лист `12_Сравнение` — три среза: всего с программатиком / без программатика + бартер / реклама без мероприятий
- Лист `15_Сводка_по_группам` — CRM + внешние доходы по категориям (программатик, бартер, ИРИ, рекомендательные системы, e-com, 47News) + расхождения в рублях и %
- Лист `16_Бухгалтерская_таблица` — структура по строкам бухгалтерии, план группы, KPI по направлениям
- Индикатор расхождения прямо в GUI: зелёный / жёлтый / красный

### Сравнение периодов
- Два произвольных файла выгрузки
- Выручка и заказы по менеджерам: Δ и %
- Топ клиентов с динамикой, новые и ушедшие
- Сгруппированные графики A vs B

### Дашборд в браузере
HTML на Chart.js — открывается одной кнопкой:
- KPI: выручка, заказы, клиенты, средний чек
- Линейный график по месяцам
- Топ-10 клиентов и менеджеров
- Пончик по отраслям

### Дополнительно
- **План менеджеров** — задаётся в интерфейсе, в отчёте: факт / план / % / статус ✅ ⚠ ❌
- **Папка-наблюдатель** — следит за папкой, запускает анализ автоматически через 5 минут после появления нового `.xlsx`
- **Toast-уведомления** — всплывающие баннеры с кнопкой "Открыть папку" по завершении
- **KPI-карточки в GUI** — CRM / Внешние / Итого / Отклонение появляются после первого запуска

## Файлы конфигурации

### `external_income.json`
Внешние доходы, которых нет в CRM: программатик, бартер, ИРИ/АНО, рекомендательные системы, e-com, 47News. Данные по каждому месяцу в рублях.

Редактируется через GUI: **⚙ Внешние доходы (программатик)** на вкладке Анализ.

Ключевые служебные поля:
- `_год` — отчётный год
- `_включать_в_аналитику` — строки, добавляемые к CRM в месячной/сезонной аналитике
- `_не_включать_в_grand_total` — закупочные и технические проводки, не входящие в итог
- `_план_группы_по_месяцам_руб` — план отдела для расчёта % выполнения
- `_план_KPI_по_месяцам_руб` — план по направлениям (Фонтанка / Доктор / СММ / Мероприятия)

### `verified_figures.json`
Верифицированные годовые итоги из бухгалтерского отчёта. Используются для сверки в листах 12 и 15.

Редактируется через GUI: **⚙ Верифицированные цифры** на вкладке Анализ.

| Поле | Описание |
|------|----------|
| `total_with_prog` | Всего с программатиком и бартером |
| `total_with_barter_no_prog` | С бартером, без программатика |
| `advertising_no_events` | Рекламная без мероприятий |
| `programmatic_external` | Программатик внешний (итог) |
| `other_external_income` | Прочие внешние доходы (ИРИ, гранты, 47) |

### `config.json`
Создаётся автоматически. Хранит последние пути к файлам, тему, планы менеджеров, выбор базовой даты.

## Входной файл (выгрузка CRM)

**Обязательные колонки:**

| Колонка | Описание |
|---------|----------|
| `Заказ` | Номер заказа |
| `Дата заказа` | Дата |
| `Выручка без НДС` | Сумма в рублях |
| `Месяц` | Формат `ММ.ГГГГ` или `ММ.ГГ` |
| `Менеджер` | ФИО менеджера |
| `Проект` | Название проекта CRM |
| `Клиент РА` / `Клиент` / `Рекламодатель` | Хотя бы одно из трёх |

**Опциональные:**

| Колонка | Описание |
|---------|----------|
| `Скидка, %` | Размер скидки — нужен для листов 17* |
| `Бартер` | `да` / пусто |
| `ДатаСуммаОплаты_` | Формат `ДД.ММ.ГГГГ СУММА` — для сверки по дате оплаты |
| `Номенклатура` | Наименование позиции |
| `Отрасли КлиентаРА` / `Отрасли Клиента` | Отрасль — нужна для листа 04 |

## Установка и запуск

### Из исходников

```bash
pip install pandas openpyxl numpy watchdog
python app.py
```

### Сборка в .exe

```bash
pip install pyinstaller
python build.py
```

Собранный файл: `dist/АналитикаЗаказов.exe`  
Файлы `external_income.json` и `verified_figures.json` включаются в `.exe` автоматически.

> При первом запуске Windows Defender может показать предупреждение — нажми «Подробнее → Всё равно запустить».
