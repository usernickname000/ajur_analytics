# Утилита «Аналитика заказов»
### Для коммерческого отдела

---

## Что делает программа

Загружает файл Excel с заказами, выполняет полный анализ
и сохраняет готовый отчёт с 13 листами:

| Лист | Содержимое |
|------|-----------|
| 00_Data_Quality | Проверка качества данных |
| 01_Месячная_статистика | Выручка и заказы по месяцам |
| 02_Топ_клиентов | ТОП-50 клиентов по выручке |
| 03_Топ_менеджеров | Статистика по менеджерам |
| 04_Отрасли | Разбивка по отраслям |
| 05_Сезонность | Выручка по сезонам |
| 06_Кварталы | Квартальная статистика + коэф. сезонности |
| 07_RFM_все | Полный RFM-анализ клиентов |
| 08_RFM_сегменты | Сводка по RFM-сегментам |
| 09_RFM_не_топ | Клиенты вне Champions/Loyal |
| 10_Распределение_заказов | Лояльность: сколько клиентов с 1/2/3+ заказами |
| 11_Расширенный_сводный | Главные показатели одним взглядом |
| 12_Сравнение | До/после фильтров |
| 13_Прогноз | Тренд и прогноз на следующий месяц |

## Установка (один раз)

### Шаг 1 — Откройте папку с утилитой в VS Code

Откройте VS Code → File → Open Folder → выберите папку `orders_utility`.

### Шаг 2 — Откройте терминал в VS Code

Меню: Terminal → New Terminal (или Ctrl + `)

### Шаг 3 — Установите библиотеки

Введите в терминале и нажмите Enter:

```
pip install -r requirements.txt
```

Дождитесь завершения установки (около 1 минуты).

---

## Запуск программы

Каждый раз для работы достаточно одной команды в терминале VS Code:

```
python app.py
```

Откроется окно программы. Дальше всё понятно из интерфейса:
1. Нажмите «Обзор...» и выберите файл Excel с заказами
2. При необходимости измените папку для сохранения отчёта
3. Нажмите «▶ Запустить анализ»
4. После завершения программа предложит открыть папку с готовым отчётом

---

## Если что-то пошло не так

**Ошибка «Критические ошибки в данных»** — программа нашла проблему в исходном файле
(отрицательная выручка, будущие даты, отсутствие обязательных колонок).
Проверьте данные в Excel.

**Ошибка «Файл не найден»** — убедитесь, что путь к файлу выбран правильно.

**Любые другие ошибки** — скопируйте текст из красного окна и передайте ответственному за утилиту.
