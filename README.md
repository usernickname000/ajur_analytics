## О проекте

Десктопная утилита для автоматической аналитики рекламных заказов. Принимает Excel-выгрузку из CRM, строит полный отчёт в Excel и интерактивный HTML-дашборд. Работает полностью офлайн.

---

## Возможности

### 📊 Анализ периода
- Месячная динамика выручки
- Топ клиентов и менеджеров
- Анализ по отраслям, сезонность, кварталы
- RFM-анализ клиентской базы
- Топ номенклатур, прогноз
- **Графики прямо в Excel** — линейный, столбчатые, круговая

### 🔄 Сравнение периодов
- Выручка и заказы по менеджерам: Δ и %
- Топ клиентов с динамикой
- Новые и ушедшие клиенты
- Сгруппированные графики A vs B

### 🌐 Дашборд в браузере
HTML-дашборд на Chart.js — открывается в браузере одной кнопкой:
- KPI: выручка, заказы, клиенты, средний чек
- Линейный график по месяцам
- Топ-10 клиентов и менеджеров
- Пончик по отраслям

### 🎯 Выполнение плана
- Плановые суммы задаются прямо в интерфейсе
- В отчёте: факт / план / % / статус ✅ ⚠ ❌

### 👁 Папка-наблюдатель
- Следит за папкой в фоне
- Через 5 минут после появления нового `.xlsx` — запускает анализ автоматически

## Установка

```powershell
git clone https://github.com/usernickname000/ajur_analytics.git
cd ajur_analytics
python -m pip install -r requirements.txt
python make_icon.py
python app.py
```

## Компиляция в .exe

```powershell
python -m PyInstaller --onefile --windowed --icon=icon.ico --add-data "icon.ico;." --name "Аналитика_Фонтанка" app.py
```

> ⚠ При первом запуске Windows Defender может показать предупреждение — нажми «Подробнее → Всё равно запустить».
